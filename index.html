FIRDA FITRIANI

Assalamualaikum Wr.Wb

Perkenalkan nama saya Firda Fitriani(Rian). Saya lahir di Jakarta 11 Desember 2001, saat ini saya berumur 17 tahun dan tengah menumpuh pedidikan di SMK-SMAK Bogor. Saya merupakan anak ke-dua dari 3 bersaudara, kakak pertama saya saat ini sudah bekerja sambil berkuliah, sedangkan adik saya masih bersekolah di taman kanak-kanak. Ayah saya merupakan seorang tentara dan ibu saya adalah seorang ibu rumah tangga. Saya dan keluarga saya tinggal di Jakarta Timur, hanya saja saya harus tinggal terpissah karena jarak antara sekolah dan rumah saya cukup jauh. Saat ini saya tinggal di kost di jalan ciheuleut.
